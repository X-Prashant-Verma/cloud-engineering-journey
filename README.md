# ☁️ Cloud & AI Infrastructure Engineering — Learning Journey

A self-directed, structured transition from a non-CS background into 
Cloud / AI Infrastructure Engineering, targeting junior roles at 
AI-first startups and product companies in India.

## 🎯 Goal
Become job-ready as a **Junior Cloud Engineer / DevOps Trainee** in 3 months,
with a clear path toward **Cloud/AI Infrastructure Engineering** (AIOps/MLOps).

## 🗺️ Roadmap Overview
Built from KodeKloud's **Cloud Engineer** + **AIOps** paths, across 6 phases:

| Phase | Focus                        | Duration   |
|-------|------------------------------|------------|
| P1    | Linux, Bash, Git, YAML       | Weeks 1–3  |
| P2    | AWS Core (EC2, S3, IAM, VPC) | Weeks 4–6  |
| P3    | Docker & Kubernetes          | Weeks 7–9  |
| P4    | Terraform & CI/CD            | Weeks 10–11|
| P5    | AIOps — Prometheus, MLflow   | Weeks 12–13|
| P6    | Capstone Projects            | Ongoing    |

## 📁 Repo Structure
Each folder contains `notes.md` (concepts in my own words) and hands-on
labs, scripts, configs, and projects built along the way.
